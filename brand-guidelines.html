03 — Motion

Animation System

Three core animation patterns. All use hardware-accelerated transform and opacity — never layout properties.

Pattern 01
Staggered Fade-in
Children appear sequentially with a fixed delay offset. Used for lists, nav items, and feature grids.
opacity 0 → 1 translateX −12px → 0 delay +80ms per child duration 400ms
Pattern 02
Slide-Up Entrance
Elements rise 24px while fading in. Triggered by IntersectionObserver at 15% visibility threshold.

Pattern 03
Scroll-Driven Scale
Background elements scale up as the user scrolls, creating a sense of depth and forward motion. Scale derived from scrollY / documentHeight.
transform: scale(1 → 1.3) passive scroll listener will-change: transform interpolate 0–100% scroll
